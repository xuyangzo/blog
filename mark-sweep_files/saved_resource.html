            acidic_base Robin R. • a month ago • edited

            In function marry(), two arguments are "man", "woman"

            we pass two objects as arguments -

            {name: "John"} and {name: "Ann"}

            so, man = {
            name: "John",
            }
            and, woman = {
            name: "Anna",
            }

            here, woman = {
            husband: man,
            }

            and, man = {
            wife: woman,
            }

            Replacing with the values, it turns,

            woman = {
            husband: "John",
            }
            man = {
            wife: "Anna",
            }

            The two objects "man" and "woman" are getting interlinked [married]

            Here a new object is being returned and assigned to family

            family = {
            father: man,
            mother: woman,
            }

            which actually means,

            family = {
            father: {
            name: "John"
            }
            mother: {
            name: "Anna"
            }
            }

            Thus, a new object family is created which has two keyed identifiers and both identifiers refer to two more objects, "man" and "woman"

            Thus here we "marry" two objects - "man" and "woman" and at the same time form one larger super set object comprising the two which is "Family"

            Hope this helps
