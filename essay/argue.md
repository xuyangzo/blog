# 好好吵架

> Posted: 01.23.2022

<Tag />

今天<span v-p>**前女友**</span>给我推荐了一个视频，是讲两个人之间如何吵架的。

[毁掉恋爱的不是吵架，是你不会吵架！进来学！！！](https://www.bilibili.com/video/BV14h411v74i)

我觉得里面说的贼有道理，所以一定要记录下来，并加入我自己的思考与总结。

## 原则一：以“我”为中心，而不是“你”

简单来说就是

- 不要你你你怎么怎么样
- 诉说自己的感受与期待

我们吵架很容易陷入一个误区，那就是使用以“你”为开头的句子。比如

```
你这么做，就是对这个家不想付出！
你就根本没把我说的话放在心上！
```

这么说会把原本的 `讲道理` 引导向 `质问/指控` 的方向，无论谁碰见都会很生气。（并且这么说同时会触犯第二个原则，这个接下来会说）

当然，你会认为这的确是对方的错啊，甚至很有可能你是在认真思考后得出的这个结论。你觉得既然是对方的问题，那你指责他难道不对吗？

然而方向对了，方法错了。

吵架本质上是沟通，只不过是一种会牵扯到暴力情绪的沟通。只要是沟通，就可以依靠说话的技巧来提高沟通的效率。

这就好比你的目标是打BOSS，那你肯定会用针对BOSS弱点的武器——这篇文章里提到的原则就是“对BOSS效果拔群”的武器。

所以你要做的不是以对方的过错为中心，而是以自己的对对方过错的感受为中心。举例：

```
我不高兴是因为我期待你能够（A 在我回来的时候把家打扫干净了），
这样我们就能（B 一起好好地看看电影什么的），
因为我希望（C 我们能够有时间好好地在一起）。
```

你要说清楚

1. 你确实因为对方的举动而不开心了
2. 对方让你不开心的原因是什么（A）
3. 你原本的期待是什么（B）
4. 骚话（C）

这样一套连击下来，不比直接暴躁吵架有效多了？

要是这个时候对方还能和你吵起来，那我建议……算了我不建议了，是你自己找的对象，又不是我硬塞给你的。

## 原则二：不要给对方“扣帽子”

简单来说就是

- 不要说你怎么样就是一个怎么怎么的人
- 不要把工作生活上的事上升到人品性格上

这一条原则其实和原则一是相辅相成的，因为吵架的时候其实经常出现：

```
你就是这么幼稚！
你就是无理取闹！
```

就比如当两个人因为擦碗要用纸巾还是抹布而吵起来时，其实只是生活习惯的不同，属于最容易去协调的一类问题。但如果想着一定要去吵赢对方，那就会不择手段。

而`扣帽子`就是不择手段的方法之一。

人是复杂的，有正面也有负面，有优点也有缺点。即便一个人做一件事的时候错了，也不能轻易地下定义。

一个人在处理人际关系的时候也许会`幼稚`，但他有可能在照顾生活很成熟，那他算是幼稚的人还是成熟的人？

**<span v-p>所以，对事不对人。</span>**

你可以说对方这件事做错了，让你很生气，但你要让他清楚，你生气是因为这件事，而不是他这个人。

当你给对方扣帽子的时候，你会强硬地把负面的形象印在他的身上，但本身做错一件事并不会导致对方直接成为一个负面的人。

吵架千万不能扩大化，扣帽子就是扩大化的一个表现，是绝对不能做的。

## 原则三：不要拿“态度”吵架

吵架的时候，人的态度绝对和心平气和时不一样。

在这个时候追求态度是完全不合理的。

而且人的态度很大程度上会随着当时心情的变化而变化。当天被老板骂了，心情变差了，可能在约会里就没那么活泼了。

即便是情绪稳定的人也多多少少会有变化。

态度重要吗？

重要。如果对一切都抱有认真的态度，也许很多时候都不会吵起来。

但因为态度吵起来的时候，真的是因为“态度”本身吗？

是的。

![pi](/images/pi.jpeg)

态度很重要，而且不仅仅是吵架前的态度很重要，吵架时的态度也很重要。

这篇文章就是在教你怎么在吵架时保持一个好的态度，也就是“好好吵架”。

那么如果是因为态度不好而吵起来的呢？我认为在这种情况下，`态度`本身应该是一个导火索，但并不是问题的核心。

比如你叫对象去把床整理一下，他没去，还跟你打哈哈，你觉得他态度不好，但其实本质是他的性格问题。

态度在绝大多数情况下反应出来的都是性格问题。

所以为什么不能拿“态度”本身去吵架，就是因为你这么说反而会让对方忽略自己可能存在的别的问题，而去纠结浮在表面上的“态度”。

他会觉得我的态度没有问题啊，现在不去整理我待会儿就去，你凭什么说我态度有问题？

你会觉得你每次这么说他都没听，这难道还不是态度的问题？

不是。不是态度问题。是你们两个的相处模式出现了问题，导致了你们两个会因为这件事出现了分歧。本质是相处模式，而不是态度。

一定要明白这一点。

## 原则四：不要主动解决TA的吐槽

这条其实很简单。

大多数男的在听到自己的女朋友在描述一个问题的时候，最先想到的可能是怎么才能去解决这个问题。

但这真的是好的解决方法吗？

设身处地地想一想，我在吐槽一件的事情的时候，会因为我自己没有想到解决方法而吐槽吗？

其实不是，我只是在吐槽事情本身，因为如果我觉得自己需要帮助，我会很清楚地告诉对方，“我不知道该怎么办”。如果我没有说，那我其实就是自己思考过，我知道应该采取什么解决方法才最好，但我还是有情绪，想要获得对方的共鸣，希望对方能够站在我这边。

如果这个时候对方告诉我应该怎么去解决这个问题，我会觉得她给予我的，和我自己想要的是不同的，也会因此而产生失望的情绪。

就连我自己都是这么想的，我为什么会觉得我的女朋友和我想的是不一样的呢？

当然，这一点涉及到的，更多的是一个人单方面向另一个人吐槽的情况，而不是两个人吵架的情况。

这个问题会导致两个人出现吵架的情况，然而并不是在吵架中会遇到的问题。

这里列出来是因为在视频里提到了。

## 原则五：先解决情绪，再解决问题

很多人会认为吵架的目的是为了解决问题，因为如果没有问题，那根本就不会吵起来。

然而更多的时候吵架只是单纯为了解决情绪，而不是解决问题。

我认为`解决问题`和`解决情绪`两者是相互独立的。99%的人做不到同时去解决这两个问题，我自然也是这99%中的一员。

在我看来，绝大多数吵架应该先以解决情绪为主，但这并不是说不去解决问题，只是因为人在被情绪左右时是听不进去任何道理的。只有在情绪平静下来后，人们才会去愿意好好地思考。

因此，先后顺序很重要，我不认为上来就尝试解决问题是好的方式。

当我们去讨论一个问题的时候，一定要处于一个能够理性地去思考的状态，否则我们的推理过程是很有可能会被情绪所影响的。

解决情绪和解决问题这两者都很重要，而且都一定要解决。

只不过不能同时解决。

好好吵架是为了前往更好的未来。
