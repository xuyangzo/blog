# 牛肉粉丝汤

> Posted: 01.08.2021

<Tag />

## 食材

- 牛肉片 x 1盒
- 龙口粉丝 x 1卷
- 八角 x 1块
- 花椒 x 5粒
- 料酒 x 2勺
- 葱花 x 1把
- 大葱 x 适量
- 姜片 x 2片
- 盐 x 适量
- 白胡椒粉 x 适量
- 咖喱粉 x 适量

## 制作

### 准备

1. 牛肉切片（或者直接买成品的牛肉片）后洗干净血水
2. 将牛肉焯水，冷水入锅，水开后撇去浮沫
3. 将粉丝在温水里泡发

### 制作

1. 锅里加水，然后加姜片、大葱、八角、花椒、料酒和牛肉
2. 大火烧开，然后转小火炖40分钟左右
3. 然后将姜、八角、大葱捞出
4. 下粉丝，然后加盐、白胡椒、咖喱粉
5. 出锅后撒上葱花
